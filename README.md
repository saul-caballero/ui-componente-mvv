# ui-componente-mvv
Componente web interactivo para mostrar misión, visión y valores mediante botones en HTML, CSS y JavaScript.


# Componente Misión, Visión y Valores
Este proyecto consiste en un componente web interactivo que muestra la **misión**, **visión** y **valores** de una empresa. Incluye tres botones que permiten alternar la información de manera dinámica.
